## Introduction to Pandas DataFrames – Replicating an Article in Markdown

This repository contains the files prepared for Assignment 4. The project demonstrates how a web article about pandas DataFrames was reproduced in a Jupyter Notebook using Markdown formatting, Python code cells, executed outputs, and an exported HTML version.

| File | Purpose |
|---|---|
| `Pandas.ipynb` | The main Jupyter Notebook containing the Markdown-formatted article, embedded image, pandas code examples, and executed outputs. |
| `dist/Pandas.html` | The exported HTML version of the final notebook for local viewing. |
| `Pandas.webp` | The image file used in the notebook and article reproduction. |
| `dist/Pandas.webp` | A copy of the image placed beside the HTML file so the image displays correctly when opening the exported HTML locally. |
